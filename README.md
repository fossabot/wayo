# Wayo

轻量的移动端Vue组件库。

## 安装
```bash
npm install wayo --save-dev
```

## 快速开始
```javascript
import Vue from 'vue';
import Wayo from 'wayo';
import './node_modules/wayo/dist/wayo.min.css';

Vue.use(Wayo);
```

## 兼容性
- Android 4+
- IOS 8+
